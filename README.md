# Heart Disease Data Cleaning & Visualization Project

## Objective
To clean, process, and visualize the Heart Disease dataset to extract meaningful insights.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Tasks Performed
- Data loading
- Missing value checking
- Duplicate removal
- Data cleaning
- Data visualization
- Insight generation

## Outcome
Successfully analyzed the Heart Disease dataset and generated visual reports showing important patterns and relationships.
